# src/

Фронтенд мини-аппы (Vue 3 + Vite). Общается с бэкендом только через
`lib/api.js` — никаких прямых обращений к БД или другим `server/`-модулям
отсюда быть не может (другой рантайм — браузер).

- `App.vue` — переключение между экраном списка и экраном чтения.
- `components/` — экраны и bottom sheet'ы (добавление книги, настройки,
  обратная связь).
- `components/icons/` — SVG-иконки (не эмодзи — те по-разному выглядят на
  разных устройствах).
- `lib/api.js` — тонкий fetch-клиент ко всем `/api/*` эндпоинтам.
- `lib/telegramUser.js` — получение telegram-пользователя (+ таймзона), с
  фолбэком для локальной разработки вне Telegram.
- `lib/devMode.js`, `lib/confirm.js`, `lib/tapOutside.js` — мелкие
  вспомогательные утилиты интерфейса.
